Bibliothèques

 

Cet outils permet d'accéder aux bibliothèques utilisées dans Tympan. On y accéde à partir de l'espace de travail.

Chaque élément métier est constitué d'un identifiant universel (ou global) unique (appelé UUID ou GUID), il permet l'échange de données métiers sans ambiguïtés sur la nature de l'élément. Cela implique aussi que 2 éléments totalement différents peuvent avoir le même nom, mais aussi que 2 éléments ayant exactement les mêmes caractéristiques peuvent être considérés comme différents s'ils ont un identifiant différent.

On distingue la bibliothèque utilisateur et la bibliothèque générale, accessibles via les onglets de la boîte de dialogue. La première est accessible à la fois en lecture et en écriture, tandis que la seconde est uniquement accessible en lecture.

Les éléments disponibles sont regroupés par type. S'ils comportent un nom, celui-ci est affiché dans la seconde colonne.

Le menu qui s'affiche lors d'un clic droit dans la fenêtre propose les choix suivants :

  • Import XML : permet d'ouvrir un fichier XML contenant des données métiers et de les enregistrer en bibliothèque, cf. l'échange de données métier en XML
  • Export XML : exporte l'élément sélectionné au format XML, cf. l'échange de données métier en XML
  • Dupliquer : effectue une copie de l'élément sélectionné et l'ajoute à la bibliothèque
  • Supprimer : supprime l'élément sélectionné de la bibliothèque, si l'utilisateur y est autorisé
  • Modeleur : affiche un modeleur pour éditer l'élément sélectionné si son type le permet (machine, construction, site ou projet)
  • Propriétés : affiche la boîte de dialogue associéé à l'élément sélectionné afin d'en éditer ses propriétés
