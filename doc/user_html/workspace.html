Espace de travail

L'espace de travail de Tympan permet d'accéder directement à chaque composant de l'application (survoler et cliquer les zones intéressés) :

 

On distingue différents types d'éléments qui composent la fenêtre principale de Tympan, pour la plupart modulables :

  • Les menus
  • Les barres d'outils
  • Les vues
  • Les fenêtres de modeleur 3D

L'application est aussi composée des éléments suivants :

  • Gestion des préférences : pour que l'utilisateur puisse personnaliser Tympan
  • Impression : des vues 3D et des tableaux
  • Bibliothèque de données métiers : bases de données utilisateur et générale
  • Echange de données métiers : pour conserver ou échanger des données métiers au format XML
  • Gestionnaire de Spectres : pour effectuer des opérations sur des spectres et les comparer
  • Gestionnaire de Maillages : pour effectuer des opérations sur des maillages et les comparer
  • Importer Tympan II : pour convertir des données métiers au format Tympan II vers Tympan 3.0
