Echange de données métier en XML

TYMPAN permet aux utilisateurs de s'échanger des données. Pour cela il utilise un format de données XML.

Tout élément d'un projet (projet, site, machine, source, calcul, ...) peut être enregistré dans un fichier au format XML.

Pour créer un fichier XML, il faut dans le menu contextuel (clic droit) d'un objet, sélectionner l'item "exporter". Un nom de fichier vous est alors demandé. Le fichier est créé dans le répertoire "echange" du répertoire userdir de l'utilisateur. Il est important pour le relire, de placer le fichier dans le répertoire "échange". En effet, la relecture du fichier nécessite la présence d'un fichier nommé "tympan.dtd" dans le répertoire du fichier à relire.
